# Contributing to Mikk

First off, thank you for considering contributing to Mikk! It's people like you that make Mikk such a great tool.

## Where do I go from here?

If you've noticed a bug or have a feature request, make sure to check our Issues tab to see if there's already an open issue. If not, go ahead and open a new one!

## Development

- **Prerequisites**: Bun, Node.js (v20+), and your favorite editor.
- **Branching Strategy**: Please create a branch from `main` and submit a Pull Request.
- **Testing**: Ensure that you run `bun run test` and `bun run lint` prior to opening a PR.

## Code of Conduct

By participating in this project, you agree to abide by the [Code of Conduct](CODE_OF_CONDUCT.md).
